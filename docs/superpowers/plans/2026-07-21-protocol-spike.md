# Protocol Spike Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Produce committed, clean-room protocol notes for the Silhouette Cameo 5 Alpha (from our own USB captures) and the GCC Puma IV (from public HPGL docs), each proven by a physical square cut.

**Architecture:** Throwaway-grade Python tooling under `tools/` captures and decodes USB traffic and drives the GCC over serial. The permanent deliverables are Markdown protocol notes under `docs/protocol/`. No Rust workspace yet — that starts in sub-project 2. The decoder and HPGL generator are TDD'd; capture and physical-cut steps use verification checklists because the machine is the test.

**Tech Stack:** Python 3.11+, `pytest`, `pyserial` (GCC serial), Wireshark + USBPcap (Windows capture) / Wireshark USB (macOS), `tshark` (payload export).

## Global Constraints

- **Clean-room:** protocol facts come ONLY from our own USB captures and public vendor/community protocol writeups. Never read, translate, or paraphrase GPL sources (`inkscape-silhouette`, `robocut`, potrace). Every documented command cites its source (capture file + packet range, or public doc URL).
- **License:** all files in this spike are Apache-2.0 (add SPDX header `# SPDX-License-Identifier: Apache-2.0` to Python files).
- **No secrets in repo:** raw `.pcapng` captures may be large/contain machine serials — commit decoded hex/notes, not raw multi-MB captures, unless a small trimmed sample. Add `tools/capture/samples/*.pcapng` to `.gitignore`; commit trimmed `*.hex` fixtures instead.
- **Decoder stays format-generic:** it splits/pretty-prints byte records on an observed terminator. Command *meanings* live in the notes, filled from observation — not hardcoded in tooling.

```python
# SPDX-License-Identifier: Apache-2.0
"""Generic USB-payload decoder: split a byte stream into records and render
them as printable ASCII. Command *meanings* are documented by hand in
docs/protocol/, not inferred here (clean-room)."""


def decode_records(hex_str: str, terminator: int = 0x03) -> list[str]:
    data = bytes.fromhex(hex_str.strip())
    out: list[str] = []
    for chunk in data.split(bytes([terminator])):
        if not chunk:
            continue
        rendered = "".join(
            chr(b) if 0x20 <= b < 0x7F else f"\\x{b:02x}" for b in chunk
        )
        out.append(rendered)
    return out
```

```python
# SPDX-License-Identifier: Apache-2.0
"""Build a minimal square cut job from the documented Cameo 5 command set and
send it over the raw USB bulk endpoint. Command syntax here mirrors
docs/protocol/silhouette-cameo5.md — update both together."""

import sys


def build_square_job(size_mm: float, steps_per_mm: float) -> list[str]:
    s = round(size_mm * steps_per_mm)
    # PU = pen up move, PD = pen down move. Origin bottom-left.
    return [
        "PU0,0",
        f"PD0,{s}",
        f"PD{s},{s}",
        f"PD{s},0",
        "PD0,0",
    ]


def send(records: list[str], terminator: bytes = b"\x03") -> bytes:
    """Serialize records to the wire byte stream. USB transport wired in the
    manual send step below (pyusb), kept out of the unit test."""
    return terminator.join(r.encode("ascii") for r in records) + terminator


if __name__ == "__main__":  # pragma: no cover — manual hardware step
    job = build_square_job(20, 20)
    payload = send(job)
    sys.stdout.buffer.write(payload)
```

```python
# SPDX-License-Identifier: Apache-2.0
"""Send an HPGL program to the GCC Puma IV over serial. Serial params per
docs/protocol/gcc-hpgl.md."""

import argparse
import serial  # pyserial

from hpgl import hpgl_square


def main() -> None:  # pragma: no cover — manual hardware step
    ap = argparse.ArgumentParser()
    ap.add_argument("--port", required=True)          # e.g. /dev/ttyUSB0
    ap.add_argument("--baud", type=int, default=9600)  # confirm from Task 6
    ap.add_argument("--size-mm", type=float, default=20.0)
    args = ap.parse_args()

    prog = hpgl_square(args.size_mm).encode("ascii")
    with serial.Serial(args.port, args.baud, timeout=5) as s:
        s.write(prog)
        s.flush()


if __name__ == "__main__":
    main()
```
